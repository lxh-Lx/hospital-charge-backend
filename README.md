# hospital-charge-backend
这是一个基于springboot来完成的门（急）诊划价收费系统是医院信息系统（HIS）基础业务的核心环节，连接医生工作站与药房、检查科室等执行部门，负责门诊患者所有诊疗费用的核算、收取、结算与票据管理。  该系统的核心业务流程为：医生开具电子处方 / 检查单 → 收费员划价核算 → 患者缴费 → 系统生成发票 → 数据同步至药房 / 检查科室。它不仅实现了门诊费用的快速准确结算，还为医院财务管理提供了完整的收费数据支撑，同时减少了人工划价的错误率，缩短了患者排队等待时间。  系统面向三类主要用户：门诊收费员、财务管理员、系统管理员，覆盖从日常收费操作到财务统计、系统维护的全流程管理。
